# 계산기 (JavaScript)
부족한 실력향상을 위해 만들어 보며 다시 학습해보기 위해 진행 :)

## 설명
- 계산하고 싶은 숫자를 입력하면 연산이 되어 결과 값이 출력되는 계산기

## 기능
```
- 계산기
  1. 계산하고 싶은 숫자 입력
  2. 사칙연산 선택
  3. 계산하고 싶은 숫자 입력
  3. 결과 값 출력 (버튼 = 클릭)
  4. 값을 초기화 하고 싶다면 버튼 C 클릭
```
